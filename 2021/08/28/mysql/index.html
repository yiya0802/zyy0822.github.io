数据库的执行：
连接数据库-查询缓存-解析器-预处理器-查询优化器-执行引擎-存储引擎
在预处理阶段会生成解析树，也就是执行器能够识别的语句，然后在优化器中选择一种最优的执行语句

mysql使用半双工的通信方式
客户端服务器不能同时向对方发送

数据库事务是DBMS执行过程中的一个逻辑单位
四大特性：
原子性，一致性，隔离性，持久性

并发出现的问题：
脏读，不可重复读:
select * from table where id=1;
update table set age=18 where id=1;
select * from table where id=1;
脏读没有提交数据然后不一致，不可重复度提交了数据然后不一致

幻读:
select * from table where id>1
insert into table values ();
select * from table where id>1
幻读是插入数据导致不一致
